# Getting Started

Welcome to Friday! This section will help you complete installation and basic setup in **5 minutes**.

---

## Install Friday

### Method 1: From Obsidian Community Plugins (Recommended)

1. Open Obsidian
2. Click the **Settings** ‚öôÔ∏è icon in the bottom left
3. Select **Community plugins**
4. Click **Browse** button
5. Search for `Friday`
6. Find the **Friday** plugin and click **Install**
7. After installation, click **Enable**

{{% hint info %}}
**Note**  
If this is your first time installing third-party plugins, you need to turn off "Restricted mode" first.
{{% /hint %}}

### Method 2: Manual Installation

1. Visit the [Friday Releases page](https://github.com/mdfriday/obsidian-friday-plugin/releases)
2. Download the latest version of `main.js`, `manifest.json`, `styles.css`
3. Find the `.obsidian/plugins/` folder in your Obsidian vault
4. Create a `friday` folder
5. Place the downloaded files in that folder
6. Restart Obsidian
7. Enable the Friday plugin in settings
