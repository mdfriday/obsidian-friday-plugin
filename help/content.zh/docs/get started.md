# 快速入门

欢迎使用 Friday！本节将帮助你在 **5 分钟** 内完成安装和基本设置。

---

## 安装 Friday

### 方式一：从 Obsidian 社区插件安装（推荐）

1. 打开 Obsidian
2. 点击左下角的 **设置** ⚙️ 图标
3. 选择 **第三方插件**
4. 点击 **浏览** 按钮
5. 搜索 `Friday`
6. 找到 **Friday** 插件并点击 **安装**
7. 安装完成后，点击 **启用**

{{% hint info %}}
**提示**  
如果是第一次安装第三方插件，需要先关闭「安全模式」。
{{% /hint %}}

### 方式二：手动安装

1. 访问 [Friday 发布页面](https://github.com/mdfriday/obsidian-friday-plugin/releases)
2. 下载最新版本的 `main.js`、`manifest.json`、`styles.css`
3. 在你的 Obsidian 仓库中找到 `.obsidian/plugins/` 文件夹
4. 创建 `friday` 文件夹
5. 将下载的文件放入该文件夹
6. 重启 Obsidian
7. 在设置中启用 Friday 插件

---

## 认识 Friday 界面

安装成功后，你会看到以下变化：

### 1. 左侧边栏 - Friday 按钮 🎲

点击可打开 **项目管理** 面板，管理你的所有发布项目。

### 2. 右侧面板 - Friday 服务

这是 Friday 的主要操作区域，包括：
- 内容选择
- 主题设置
- 预览生成
- 发布选项

### 3. 笔记顶部 - 地球图标 🌐

点击可 **快速分享** 当前笔记到网上。

### 4. 设置页面 - Friday 设置

在 Obsidian 设置中，你可以找到 Friday 的配置选项：
- License 激活
- 同步设置
- 发布设置
