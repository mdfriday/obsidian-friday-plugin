# 常见问题

最常见问题及解决方案的汇总。

---

## 安装与激活

### Q：安装后找不到 Friday 设置？

**解决方案**：
1. 确认插件已安装：设置 → 第三方插件 → 已安装插件
2. 确认插件已启用：开关已打开
3. 重启 Obsidian

### Q：License 激活失败？

**检查清单**：
- [ ] License Key 格式正确（`MDF-XXXX-XXXX-XXXX`）
- [ ] 没有多余的空格
- [ ] 网络连接正常
- [ ] License 未过期
- [ ] 设备数量未超限

### Q：如何查看 License 状态？

打开设置 → Friday，License 区域显示：
- License Key（已遮掩）
- 有效期至
- 套餐类型
- 存储用量

## 同步问题

### Q：同步不工作？

**排查步骤**：
1. 确认 License 已激活
2. 检查网络连接
3. 查看状态栏同步图标
4. 尝试手动同步：命令面板 → "Sync: Pull from Server"

### Q：加密密码丢失了？

{{% hint danger %}}
**这是严重问题**

如果密码确实丢失且无法恢复：
1. 当前设备上的本地数据不受影响
2. 云端加密数据无法恢复
3. 需要重新上传本地数据（会覆盖云端）
{{% /hint %}}

**预防措施**：
- 保存在密码管理器中
- 纸质备份
- 发送到自己的邮箱

### Q：两台设备同步冲突？

Friday 会尝试自动合并。如果自动合并失败：
1. 两个版本都会保留
2. 手动选择保留哪个
3. 建议避免同时编辑同一文件

### Q：同步很慢？

可能原因：
- 首次同步需要传输所有数据
- 网络速度限制
- 图片或附件较多

建议：
- 使用稳定网络同步
- 大文件需要更多时间
- 首次同步请耐心等待

## 发布问题

### Q：预览构建失败？

**常见原因及解决方案**：

| 问题 | 解决方案 |
|------|----------|
| 未选择内容 | 右键文件夹/文件 → Publish to Web |
| 主题下载失败 | 检查网络，尝试更换下载服务器 |
| Markdown 语法错误 | 检查笔记内容，修复错误 |
| 图片路径错误 | 确保图片在仓库中，使用相对路径 |

### Q：预览页面打不开？

**检查**：
1. 查看预览链接中的实际端口
2. 确认防火墙允许该端口
3. 尝试重新生成预览

### Q：Netlify 发布失败？

**检查清单**：
- [ ] Access Token 正确
- [ ] Project ID 正确
- [ ] Token 有足够权限
- [ ] Netlify 账号正常

### Q：FTP 上传失败？

**排查步骤**：
1. 使用「测试连接」验证
2. 检查用户名/密码
3. 检查远程目录权限
4. 尝试勾选「忽略证书」

### Q：发布后样式丢失？

**原因**：SitePath 配置错误

**解决方案**：
1. 确认 SitePath 与实际部署路径一致
2. 重新生成预览并发布
3. 清除浏览器缓存

## 主题问题

### Q：主题加载不出来？

**解决方案**：
1. 检查网络连接
2. 尝试更换下载服务器：设置 → Friday → 下载服务器
3. 等待几秒后重试

### Q：切换主题后效果不对？

不同主题对内容结构有不同要求：
1. 下载主题样例了解正确结构
2. 调整内容以适应主题
3. 某些主题只适合特定类型的内容

### Q：如何获取更多主题？

1. 在主题选择器中浏览
2. 使用标签筛选
3. 关注 Friday 更新获取新主题

## 性能问题

### Q：Obsidian 变慢了？

**可能原因**：
1. 预览文件占用太多空间
2. 同步文件过多

**解决方案**：
1. 清理不需要的预览历史
2. 检查同步文件数量
3. 重启 Obsidian

### Q：预览文件占用太多空间？

**清理方法**：
1. 点击 Friday 按钮打开项目管理
2. 选择项目 → 清除历史
3. 或手动删除 `.obsidian/plugins/friday/preview/` 目录

## 移动端问题

### Q：移动端支持发布吗？

目前移动端只支持：
- ✅ License 激活
- ✅ 同步功能

暂不支持：
- ❌ 预览生成
- ❌ 网站发布

由于移动平台限制。未来版本可能会添加支持。

### Q：移动端同步正常吗？

是的！移动端同步与桌面端完全一致：
- 自动同步
- 端到端加密
- 实时更新
