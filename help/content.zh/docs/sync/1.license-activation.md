# License 激活

License Key 是你使用 Friday 同步和发布功能的凭证。本指南将引导你完成激活流程。

---

## 获取 License Key

### License Key 格式

Friday 的 License Key 格式为：

```
MDF-XXXX-XXXX-XXXX
```

示例：`MDF-A1B2-C3D4-E5F6`

### 如何获取

1. **免费试用**：访问 [MDFriday 网站](https://mdfriday.com) 申请
2. **购买**：选择适合你需求的套餐

---

## 激活步骤

### 步骤 1：打开设置

1. 打开 Obsidian
2. 点击左下角的 **设置** ⚙️ 图标
3. 在左侧菜单找到 **Friday**

### 步骤 2：输入 License Key

1. 找到 **License** 区域的输入框
2. 输入你的 License Key（如 `MDF-A1B2-C3D4-E5F6`）
3. 系统会自动转换为大写

### 步骤 3：点击激活

1. 点击 **Activate** 按钮
2. 等待验证（通常几秒钟）
3. 成功后，你的 License 信息会显示出来

---

## 激活成功后

激活成功后，你会看到：

| 信息 | 说明 |
|------|------|
| License Key | 显示为遮掩的 `MDF-••••-••••-XXXX` |
| 有效期至 | License 过期日期 |
| 套餐类型 | Free / Pro / Team 等 |
| 存储用量 | 已用空间 / 总空间 |

## 常见问题

### Q：License Key 无效？

检查：
- 格式是否正确（`MDF-XXXX-XXXX-XXXX`）？
- 有没有多余的空格？
- 字母是否正确（注意：0 vs O，1 vs l）？

### Q：激活失败？

可能原因：
- 网络连接问题
- License 已过期
- 设备数量已达上限

### Q：如何在新设备上使用？

每个 License 支持一定数量的设备。在新设备上用相同的 License Key 激活，然后输入加密密码即可同步。
