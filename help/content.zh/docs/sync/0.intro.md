# 安全同步

Friday 的同步功能让你的笔记在多设备之间 **安全且实时** 地同步。

---

## 核心特性

{{% columns %}}

### 🔒 端到端加密

你的笔记在上传前就已加密。只有你的密码能解密它们。即使服务器管理员也无法查看你的内容。

===

### 📱 多设备同步

电脑、手机、平板——只要安装了 Friday，你的笔记就能实时同步。

===

### ☁️ 本地优先

笔记始终存储在你的设备上。云端只是同步的桥梁。即使离线，你也能正常工作。

{{% /columns %}}

---

## 同步原理

Friday 使用 **CouchDB** 协议进行同步，这是一种成熟的分布式数据库技术：

1. **本地存储**：笔记保存在设备的 IndexedDB 中
2. **变更检测**：自动检测文件的新增、修改、删除
3. **增量同步**：只同步变化的部分，节省带宽
4. **冲突处理**：智能处理多设备同时编辑的情况

---

## 开始使用同步

要使用同步，你需要：

1. **获取 License Key**
2. **在第一台设备上激活**
3. **在其他设备上激活并同步**

{{% hint warning %}}
**重要**  
首次激活时，系统会生成一个 **加密密码**。请务必保存它！后续所有设备都需要这个密码来解密云端数据。
{{% /hint %}}
