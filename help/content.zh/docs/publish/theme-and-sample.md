# 主题与样例

主题决定了你网站的外观和功能。Friday 提供丰富的主题选择，让你的内容以最佳方式呈现。

---

## 选择主题

### 打开主题选择器

在右侧面板的「主题」区域，点击 **Change Theme** 按钮。

### 主题选择器界面

主题选择器包含：

| 区域 | 功能 |
|------|------|
| **搜索框** | 按名称、作者、标签搜索 |
| **标签筛选** | 按类型筛选（Blog、Book、Page 等） |
| **主题卡片** | 显示预览图、名称、描述 |
| **Live Demo** | 预览主题效果（鼠标悬停时显示） |
| **Use** 按钮 | 选择该主题 |

### 主题类型

| 标签 | 使用场景 | 示例主题 |
|------|----------|----------|
| **Book** | 文档、知识库、教程 | Obsidian Book |
| **Blog** | 个人博客、文章 | Hugo Blog |
| **Page** | 单页、简历、落地页 | Note |
| **Gallery** | 图片展示、作品集 | Gallery |
| **Business** | 公司网站、产品介绍 | Corporate |

---

## 自动主题选择

Friday 会根据你的内容类型推荐主题：

| 内容类型 | 推荐主题 |
|----------|----------|
| 文件夹（多个文件） | Book（用于文档/博客） |
| 单个 Markdown 文件 | Note（用于单页） |

你总是可以手动更改。

## 下载样例

部分主题提供 **样例内容**，帮助你快速了解功能和最佳实践。

### 如何下载

1. 选择一个主题
2. 如果主题有样例，会出现 **Download Sample** 按钮
3. 点击按钮，样例内容下载到你的仓库

### 样例位置

样例下载到 Obsidian 仓库的 `MDFriday` 文件夹：

```
你的仓库/
└── MDFriday/
    └── theme-name/     ← 样例内容
        ├── content/
        ├── static/
        └── ...
```

### 使用样例

1. 下载后，右键 `MDFriday/theme-name` 文件夹
2. 选择 **Publish to Web**
3. Friday 自动检测内容结构
4. 生成预览查看效果

这是学习主题功能的最佳方式！

## 主题特性

### Obsidian 风格主题

标记为 `Obsidian` 的主题支持完整的 Obsidian 渲染：

- ✅ Obsidian 原生样式
- ✅ 插件渲染（如 Dataview）
- ✅ 自定义 CSS 片段
- ✅ 主题样式

### 标准主题

没有 `Obsidian` 标签的主题使用轻量渲染：

- ✅ 构建速度快
- ✅ 支持标准 Markdown
- ✅ 体积更小
- ⚠️ 部分 Obsidian 特有语法可能不支持

## 主题配置

### 基本配置

大多数主题通过以下方式配置：

1. **站点名称**：显示在网站标题
2. **Site Path**：影响 URL 结构
3. **高级设置**：Google Analytics、Disqus 等

### 特定主题配置

部分主题支持额外配置，通常通过内容文件夹中的 `config` 文件或 frontmatter 实现。

下载主题样例查看具体配置方法。

## 常见问题

### Q：切换主题后效果不对？

不同主题对内容结构有不同要求：
1. 确保你的内容结构符合主题要求
2. 参考主题样例了解正确结构
3. 重新生成预览

### Q：找不到合适的主题？

1. 使用搜索和标签筛选
2. 单页内容查看 "Page" 标签的主题
3. 下载样例了解主题效果
4. 在 GitHub 上提交主题需求

### Q：样例下载失败？

检查：
1. 网络连接是否正常？
2. 下载服务器是否可用？
3. 尝试切换下载服务器（在设置中）

### Q：如何自定义主题？

目前不支持自定义主题，但你可以：
1. 使用 CSS 片段微调样式
2. 在 GitHub 上提交主题改进建议
3. 贡献你自己的主题（欢迎 PR）
