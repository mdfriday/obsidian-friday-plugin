# 右侧面板

右侧面板是 Friday 的主要操作区域，让你完全控制网站的构建和发布。

---

## 打开面板

两种方式打开右侧面板：

### 方式一：右键菜单

1. 在文件列表中，右键点击一个 **文件夹** 或 **Markdown 文件**
2. 选择 **Publish to Web**

### 方式二：命令面板

1. 按 `Ctrl/Cmd + P` 打开命令面板
2. 搜索 "Friday" 或 "Publish"
3. 选择相应的命令

---

## 面板区域说明

右侧面板从上到下包含以下部分：

### 1️⃣ 多语言内容

这个区域管理你的网站内容来源。

| 元素 | 说明 |
|------|------|
| **内容路径** | 显示你选择的文件夹或文件名 |
| **语言选择** | 为每个内容设置语言（中文、英文等） |
| **Default** 标签 | 表示主要语言版本 |
| **Clear** 按钮 | 移除所有已选内容 |
| **×** 按钮 | 移除单个语言版本（多语言时显示） |

**多语言支持：**

Friday 支持多语言网站。如果你有多种语言的内容：

1. 首先添加主要语言内容文件夹
2. 右键其他语言文件夹并选择 "Publish to Web" 来添加它们
3. 为每个内容设置正确的语言

### 2️⃣ 站点名称

输入你的网站标题，显示在网站头部。

```
站点名称: [我的博客          ]
```

### 3️⃣ 站点资源

管理静态资源（图片、CSS、JS 等）。

- 显示当前设置的资源文件夹
- 点击 "Clear Assets" 移除

**设置站点资源：**

1. 在文件列表中右键点击一个文件夹
2. 选择 **Set as Site Assets**
3. 该文件夹中的文件会被复制到 `/static` 目录

### 4️⃣ 高级设置

点击展开更多配置选项：

| 设置 | 说明 | 示例 |
|------|------|------|
| **Site Path (SitePath)** | 网站的 URL 路径 | `/`、`/blog`、`/s/xxx/yyy` |
| **Site Password** | 设置访问密码 | 留空表示公开 |
| **Google Analytics ID** | 流量分析 | `G-XXXXXXXXXX` |
| **Disqus Shortname** | 评论系统 | `your-site-name` |


### 5️⃣ 主题选择

选择网站的视觉风格。

- **当前主题名称**：显示已选主题
- **Change Theme** 按钮：打开主题选择器
- **Download Sample** 按钮：下载主题示例内容（如果有）


### 6️⃣ 预览

发布前预览网站。

- **Generate Preview** / **Regenerate Preview** 按钮
- 进度条显示构建进度
- 预览链接（构建后显示）
- **Export Site** 按钮（下载为 ZIP）


### 7️⃣ 发布

将网站发布到互联网。

- **发布方式** 下拉框：选择发布平台
- 平台配置区域（根据选择变化）
- **Publish** 按钮

## 典型工作流程

### 发布文件夹

1. 右键内容文件夹 → **Publish to Web**
2. 输入站点名称
3. （可选）设置站点资源文件夹
4. 选择主题
5. 点击 **Generate Preview** → 检查效果
6. 选择发布方式 → 点击 **Publish**

### 发布单个文件

1. 右键 Markdown 文件 → **Publish to Web**
2. 输入站点名称
3. 选择适合单页的主题（如 Note）
4. 生成预览 → 发布

### 创建多语言网站

1. 准备多语言内容文件夹（如 `content`、`content.zh`）
2. 右键主文件夹 → **Publish to Web**
3. 右键其他语言文件夹 → **Publish to Web**（作为新语言添加）
4. 为每个内容设置正确的语言
5. 生成预览 → 发布

## 智能文件夹检测

Friday 自动检测 Hugo 风格的文件夹结构：

```
my-site/
├── content/      → 自动检测为英文内容
├── content.zh/   → 自动检测为中文内容
└── static/       → 自动检测为站点资源
```

当你右键点击这种结构的父文件夹时，Friday 会自动配置所有内容和资源。

## 常见问题

### Q：内容显示不正确？

检查：
1. 选择的文件夹/文件正确吗？
2. 语言设置正确吗？
3. 主题适合你的内容类型吗？

### Q：如何清除当前配置？

点击「多语言内容」区域的 **Clear** 按钮，移除所有已选内容。

### Q：为什么看不到某些设置？

部分高级设置需要展开。点击「高级设置」旁边的 ▶ 图标来展开。
